# choice-two (どれにする) 仕様書 v1_0
## ゴール
2〜4枚の絵カードから1つを選んで「これにする」を伝える選択ボードのChrome拡張。ことばが出にくい子のAAC(意思表示)支援。
## 絶対制約
外部API・通信なし/chrome.storage.localのみ/権限storageのみ/MV3・TS・Vite/UIはpopup内で完結。医療・診断をうたわない。
## 機能
選択肢カード(絵文字+ことば)を保護者が2〜4枚に設定・編集・削除/子供が1枚タップ→大きく表示して「これにする」を確定(やさしい演出)/選択肢セットを複数保存・切替(おやつ/あそび 等)/保護者・子供モードを簡易PINで保護/全状態を保存・復元/i18n ja-en/無料はセット1つ・最大3枚、Premium($3買い切り7日トライアル)でセット無制限・最大6枚+よく使うセット。
## 完了条件
npm run build成功・dist生成・_locales ja/en・icons16/48/128・release/choice-two.zip生成。
